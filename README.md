gedit3-codefolding
==================

This plugin adds code folding support to gedit 3.x. It supports several (meaning more than 1) languages. Note that this plugin is <b>not</b> compatible with gedit 2.x.

##Suported languages
- C/C++
- CSS
- Graphviz Dot
- Java
- Javascript
- JSON
- Lua
- Objective-C
- Perl
- Prolog
- R
- sh
- Scheme
- SQL
- XML
